# Security Policy

## Data

This project uses **synthetic data only**, generated at runtime or via
`src/data_generator.py`. It contains no real IBM, Kyndryl, SAP, client,
personal or confidential data of any kind.

## Secrets

The application requires no API keys, credentials or secrets. Do not commit
local environment files or Streamlit secrets.

## Reporting

If you notice a security issue in the code or dependencies, please open a
GitHub issue or contact the maintainer via the profile links.
