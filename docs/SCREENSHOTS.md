# Screenshots Guide

After deployment, capture these recruiter-facing screenshots:

1. Executive Overview — KPI overview and weekly ticket volume
2. SLA Breach Risk — ML breach probability and escalation queue
3. Root Cause Analysis — Pareto and recurring root causes
4. Service Health — service health scorecard
5. PMO Action Plan — recommendations and export buttons

Add the image files to an `assets/screenshots` folder and embed the strongest screenshots in the README.
